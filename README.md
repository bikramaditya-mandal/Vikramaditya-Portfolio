# Dr. Bikramaditya Mandal — Portfolio (Next.js 14 + Tailwind CSS)

## Stack
- **Next.js 14** (App Router) — SSR, file-based routing, built-in SEO via Metadata API
- **TypeScript** — full type safety
- **Tailwind CSS** — utility-first styling
- **Google Fonts** — Outfit + DM Mono (loaded via CSS @import)

## Setup

### 1. Install dependencies
```bash
npm install
```

### 2. Copy your assets
Place all images from your old `assets/` folder into `public/assets/`.

### 3. Run dev server
```bash
npm run dev
```
Open http://localhost:3000

### 4. Build for production
```bash
npm run build
npm start
```

## Deploying to GitHub Pages / Vercel

### Vercel (Recommended — easiest)
1. Push this folder to a GitHub repo
2. Go to vercel.com → Import project
3. Vercel auto-detects Next.js — click Deploy
4. Done. Automatic HTTPS + CDN.

### GitHub Pages (static export)
Add to `next.config.js`:
```js
const nextConfig = {
  output: 'export',
  images: { unoptimized: true },
}
```
Then run `npm run build` — the `out/` folder is your static site.

## SEO Improvements vs Old HTML
- Each page has its own `<Metadata>` with `title`, `description`, `keywords`, `openGraph`
- Images use Next.js `<Image>` with automatic WebP conversion + lazy loading
- Semantic HTML throughout
- `robots.txt` and `sitemap.xml` auto-generated by Next.js on build
